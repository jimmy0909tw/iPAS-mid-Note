📘 階段四：機器學習技術深化

涵蓋監督式與非監督式學習、深度學習架構、模型選擇與效能評估三大模組，強化模型理解與判斷能力

🔹 監督式 vs 非監督式學習

原理：監督式學習使用有標籤資料進行預測，非監督式學習則探索資料結構。

  • 監督式學習：分類、迴歸任務。
    比喻：像是「有答案的考卷」。
  • 非監督式學習：聚類、降維任務。
    比喻：像是「自由分組」活動。

  • (A) 客戶信用風險分類
  • (B) 預測未來銷售額
  • (C) 找出資料中的潛在群集 ✅
  • (D) 判斷影像是否為貓或狗

🔹 深度學習架構

原理：深度學習使用多層神經網路處理複雜資料，如影像、語音、序列。

  • CNN：卷積神經網路，擅長影像辨識。
    比喻：像機器的「眼睛」。
  • RNN：遞迴神經網路，擅長序列資料。
    比喻：像有記憶的腦袋。
  • ResNet：使用殘差連接避免梯度消失。
    比喻：像加了捷徑的迷宮。
  • Inception：多尺度特徵提取。
    比喻：像多角度觀察物體。

  • (B) Inception ✅

  • (D) VGG19

🔹 模型選擇與效能評估

原理：根據任務選擇合適模型，並使用指標評估其效能。

  • Accuracy：整體預測正確率。
    公式： (TP + TN) / (TP + TN + FP + FN)
  • Precision：預測為正的準確性。
    公式： TP / (TP + FP)
  • Recall：真實正例被抓到的比例。
    公式： TP / (TP + FN)
  • F1-score：精準與召回的平衡。
    公式： 2 × (Precision × Recall) / (Precision + Recall)
  • R²：迴歸模型的擬合程度。
    公式： 1 - (SS_res / SS_tot)

  • (A) R² ✅
  • (B) F1-score
  • (C) AUC
  • (D) Precision
