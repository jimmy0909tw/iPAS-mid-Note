📘 階段三：資料處理與模型訓練

涵蓋資料前處理、模型訓練與優化三大模組，強化實作能力與效能判斷

🔹 資料前處理

原理：清洗與轉換資料，使其適合模型訓練，避免偏誤與不穩定。

  • Z-score 標準化：使特徵平均值為 0，標準差為 1。
    公式： Z = (X - μ) / σ
    比喻：像把不同單位的成績轉成「標準分」。
  • Min-Max 正規化：將特徵縮放至 0~1 區間。
    比喻：像把不同高度的人都壓縮到比例尺。
  • 資料重抽樣：平衡類別樣本數，避免模型偏向多數類別。

📌 應用題：

若數據中出現特徵尺度差異極大（如年齡 0–100、收入 0–1,000,000），應採用哪種預處理方式？

  • (A) 移除尺度較小的欄位
  • (B) Z-score 標準化 ✅
  • (C) 對所有數值欄位加常數
  • (D) 不需處理，模型會自動調整

🔹 模型訓練

原理：使用訓練資料建立模型，使其學習資料間的規律。

  • 資料分割：將資料分為訓練集與測試集，避免過度擬合。
  • Logistic Regression：用於二元分類，輸出機率值。
  • 模型評估：使用 Accuracy、Precision、Recall、F1-score 等指標。

📌 應用題：

某公司希望根據顧客年齡與消費金額預測是否為高價值顧客，以下哪段程式碼順序正確？

  • (A) a → b → c → d
  • (B) c → a → b → d ✅
  • (C) b → c → a → d
  • (D) d → a → c → b

🔹 模型優化

原理：提升模型效能與穩定性，避免過度擬合與欠擬合。

  • Adagrad：自動調整每個參數的學習率，適合稀疏資料。
  • 交叉驗證：多次分割資料驗證模型穩定性。
  • 正則化：限制模型複雜度，避免過度擬合。

📌 應用題：

哪一種優化器會自動調整每個參數的學習率，特別適用於稀疏資料？

  • (A) Momentum
  • (B) Adagrad ✅
  • (C) Adam
  • (D) SGD
